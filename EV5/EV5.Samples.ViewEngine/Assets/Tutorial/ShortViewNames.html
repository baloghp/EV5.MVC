Shortening View names
It is possible to specify a resource name for any code view via the MarkupName Attribute class.
This way you can give any name to your code view the ViewEngine will always look for the corresponding markup under the name as specified in the MarkupName attribute. Thus you can use a shorter view name than the resource name. Like: 
[EmbeddedView("eve-ShortViewNames.Banner.html")]
    [MarkupName("Assets.Views.Sample.ShortViewNames.Banner.html")]
    public class Banner : DataBindingView<Models.Banner>
    {
        public override void ProcessView(System.Web.Mvc.ViewContext viewContext)
        {
        
        }
